# 알고리즘 학습 저장소

알고리즘과 자료구조를 학습하고 구현한 코드를 정리하는 레포지토리입니다.

## 폴더 구조

```
Algorithm/
├── chapter1/     # 이진 탐색 & 탐색 알고리즘
├── chapter2/     # 정렬 알고리즘
├── chapter3/     # 재귀
├── chapter4/     # 퀵 정렬
├── chapter5/     # 해시 테이블
├── chapter6/     # 너비 우선 탐색 (BFS)
├── chapter7/     # 트리 순회 & 허프만 코딩
├── chapter8/     # 트리 자료구조
├── chapter9/     # 다익스트라 알고리즘
├── chapter10/    # 그리디 알고리즘
├── chapter11/    # 동적 프로그래밍
└── chapter12/    # K-최근접 이웃 (KNN)
```

## 기술 스택

- **언어**: Python 3.12+
- **주요 개념**: 
  - 탐색 알고리즘 (이진 탐색, BFS, DFS)
  - 정렬 알고리즘 (선택 정렬, 퀵 정렬)
  - 자료구조 (해시 테이블, 트리)
  - 그래프 알고리즘 (다익스트라)
  - 알고리즘 패러다임 (그리디, 동적 프로그래밍, 분할 정복)

## 학습 목표

- 기본적인 알고리즘과 자료구조 이해
- 시간 복잡도와 공간 복잡도 분석
- 문제 해결 능력 향상
- 효율적인 코드 작성 능력 배양
